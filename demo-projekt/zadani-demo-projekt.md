# Demo projekt

Cílem je vytvořit kompletní dokumentaci a návrh pro novou funkci, kterou chceme přidat do pomyslné bankovní mobilní aplikace.

## Úkoly
0. Zjisti, o co jde v demo projektu.
1. Vytvoř shrnutí cíle projektu a hlavních funkcí.
2. Vytvoř epik podle šablony `./templates/epic.template.md`
9. Vytvoř glosář pojmů, které budou použity v dokumentaci
3. Uprav `.github/copilot-instructions.md` pro ukládání do `demo-projekt/epics/`
4. Vytvoř první user story podle šablony `./templates/user-story.template.md`
5. Uprav prompt v `.github/prompts/nova-user-story.md` a automatizuj vytváření dalších user stories
6. Vytvoř testovací scénáře pro každou user story ve formátu Gherkin podle šablony `./templates/test-case.template.md`
7. Nainstaluj si extension Mermaid Chart a vygeneruj sekvenční diagramy pro každou user story
8. Vygeneruj ER diagram pro znázornění datového modelu nové funkce
9. Nainstaluj si extension OpenAPI Editor a navrhni OpenAPI specifikaci pro novou funkci
10. Vygeneruj testovací datové sady ve formátu JSON
11. Validuj dokumentaci pomocí šablony DoD `./templates/dod.template.md`
12. Nech Copilota zduplikovat šablonu DoD do `demo-projekt/dod/` a vyplnit ji podle tvé dokumentace
13. Doplň chybí části dokumentace a nech AI aktualizovat zduplikované DoD
14. Rozděl zbývající práci do tiketů
