# Demo projekt

Cílem je vytvořit kompletní dokumentaci a návrh pro novou funkci, kterou chceme přidat do pomyslné bankovní mobilní aplikace.

## Úkoly
0. Zjisti, o co jde v demo projektu.
1. Vytvoř shrnutí cíle projektu a hlavních funkcí.
2. Vytvoř epik podle šablony `./sablony/epic.sablona.md`
3. Vytvoř glosář pojmů, které budou použity v dokumentaci
4. Uprav `../.github/copilot-instructions.md` pro ukládání do `demo-projekt/epics/`
5. Vytvoř první user story podle šablony `./sablony/user-story.sablona.md`
6. Uprav prompt v `../.github/prompts/nova-user-story.md` a automatizuj vytváření dalších user stories
7. Vytvoř testovací scénáře pro každou user story ve formátu Gherkin podle šablony `./sablony/test-case.sablona.md`
8. Nainstaluj si extension Mermaid Chart a vygeneruj sekvenční diagramy pro každou user story
9. Vygeneruj ER diagram pro znázornění datového modelu nové funkce
10. Nainstaluj si extension OpenAPI Editor a navrhni OpenAPI specifikaci pro novou funkci
11. Vygeneruj testovací datové sady ve formátu JSON
12. Validuj dokumentaci pomocí šablony DoD `./sablony/dod.sablona.md`
13. Nech Copilota zduplikovat šablonu DoD do `demo-projekt/dod/` a vyplnit ji podle tvé dokumentace
14. Doplň chybějící části dokumentace a nech AI aktualizovat zduplikované DoD
15. Rozděl zbývající práci do tiketů
